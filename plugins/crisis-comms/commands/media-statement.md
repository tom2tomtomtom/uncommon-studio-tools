# /media-statement

Create a media statement with Q&A preparation.

## What this does
Drafts a press statement and prepares you for tough journalist questions with suggested answers.

## When to use
- Media calling about an issue
- Need to get ahead of a story
- Preparing spokesperson for interviews

## How to use
Share:
- What the situation is
- What's public vs. confidential
- Your organization's position
- Who might be the spokesperson

## Example
```
/media-statement

A journalist is asking about the data breach we disclosed last week. They want to know:
- How many customers affected
- What data was exposed
- What we're doing about it

We can share: 10,000 customers, email addresses only, free credit monitoring offered.
Spokesperson: Our CTO
```

## What you'll get
1. **Official statement** - Ready to send
2. **Q&A document** - Anticipated questions with suggested answers
3. **Bridging phrases** - How to redirect difficult questions
4. **What NOT to say** - Common pitfalls to avoid
5. **Follow-up talking points** - If the interview continues
