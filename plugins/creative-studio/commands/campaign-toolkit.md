# /campaign-toolkit

Build a full campaign visual toolkit with heroes and social suite.

## What this does
Generates a complete set of campaign visuals from a single brief — hero images for web and print, plus a social media suite sized for each platform. Maintains visual consistency across all formats while optimizing composition for each aspect ratio.

## When to use
- Launching a campaign and need visuals across multiple channels
- Building a pitch deck with realistic campaign mockups
- Need consistent imagery for web, social, email, and display ads
- Want to prototype a full visual campaign before investing in production

## How to use
Share:
- Campaign concept and key message
- Target audience
- Brand guidelines or visual style
- Channels needed (web, Instagram, LinkedIn, email, etc.)
- Number of hero concepts to explore (1-3)
- Any mandatory elements (product shots, specific scenes, text overlay space)

## Example
```
/campaign-toolkit

Campaign: "Start Fresh" — January wellness push for a meal delivery service
Message: Healthy eating doesn't have to be hard
Audience: Busy professionals 28-45
Style: Bright, clean, overhead food photography with natural ingredients
Channels: Website hero, Instagram feed + stories, LinkedIn, email header
Concepts: 2 hero directions to choose from
Requirements: Leave space for text overlay on heroes, food should look approachable not intimidating
```

## What you'll get
1. **Hero concepts** — 1-3 key visual directions at 16:9 for web
2. **Social suite** — Each hero adapted for Instagram (1:1 + 9:16), LinkedIn (3:2), and other requested formats
3. **Email header** — Optimized for email clients at 3:2
4. **Visual consistency guide** — How the concept translates across formats
5. **Art direction notes** — Composition decisions, what to preserve when resizing, and production recommendations
