# /spec-sheet

Create technical specification sheets for production deliverables.

## What this does
Generates detailed technical specs for all deliverables, ensuring everyone knows exactly what's being delivered and in what format.

## When to use
- Briefing post-production vendors
- Defining deliverables for contracts
- Creating delivery checklists
- Ensuring platform compatibility

## How to use
Share:
- Type of assets (video, photo, audio)
- Platforms/placements where they'll be used
- Any special requirements (accessibility, legal, etc.)
- Versioning needs (languages, aspect ratios, lengths)

## Example
```
/spec-sheet

Video campaign delivering to:
- TV broadcast (national)
- YouTube pre-roll
- Instagram (feed and stories)
- TikTok
- Client website

Need Spanish language versions for TV and social.
Must include closed captions.
:30 and :15 cutdowns.
```

## What you'll get
- **Master Deliverables List** - All assets at a glance
- **Video Specs** - Resolution, codec, frame rate, audio specs per platform
- **Versioning Matrix** - All combinations needed
- **Accessibility Requirements** - Captions, audio description
- **File Naming Convention** - Standardized naming
- **Delivery Format** - How files should be packaged
- **QC Checklist** - What to verify before delivery
