# /shot-list

Generate a detailed shot list for video or photo productions.

## What this does
Creates an organized shot list that ensures you capture everything needed on shoot day, with setup details and technical notes.

## When to use
- Pre-production planning
- Director/photographer briefing
- Estimating shoot day schedule
- Ensuring creative coverage

## How to use
Share:
- What you're shooting (concept/storyboard summary)
- Type of production (video/photo)
- Talent involved
- Products or props
- Location(s)
- Any specific shots the creative requires

## Example
```
/shot-list

Photo shoot for athletic apparel brand. Shooting at outdoor track location.
2 models - male and female athlete.
Need: hero campaign shots, product detail shots, lifestyle/action shots.
Products: running shoes, performance tees, shorts.
Using for: website, social, maybe OOH.
```

## What you'll get
- **Shot List Table** - Numbered shots with descriptions
- **Setup Groups** - Organized by lighting/location setup
- **Coverage Notes** - Wide/medium/close for each setup
- **Talent Requirements** - Who's in each shot
- **Product/Prop Requirements** - What's needed when
- **Technical Notes** - Special considerations
- **Time Estimates** - Rough timing per setup
