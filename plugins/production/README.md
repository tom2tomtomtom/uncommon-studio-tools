# Production Plugin

Turn Claude into your production expert for video, photo, print, and digital asset creation.

## What's Included

### Skills
- **Production Expertise** - Deep knowledge of production workflows, vendor management, budgeting, and quality control

### Commands
- `/production-brief` - Create comprehensive production briefs for video, photo, or print projects
- `/shot-list` - Generate detailed shot lists for video or photo shoots
- `/spec-sheet` - Create technical specification sheets for deliverables

## Use Cases

- Planning video shoots (commercials, social content, brand films)
- Organizing photo shoots (product, lifestyle, campaign imagery)
- Managing print production (OOH, collateral, packaging)
- Creating asset delivery specifications
- Vendor selection and management
- Budget tracking and cost estimation
- Post-production workflow management

## Getting Started

1. Install the plugin in your Claude Cowork workspace
2. Use `/production-brief` to start planning a production
3. Let Claude help with shot lists, specs, and vendor coordination

## MCP Connectors

This plugin can connect to:
- **Slack** - Share production updates and coordinate with teams
- **Notion** - Manage production schedules and documentation
- **Google Drive** - Access reference materials and deliverables
- **Frame.io** - Review and approve video content
