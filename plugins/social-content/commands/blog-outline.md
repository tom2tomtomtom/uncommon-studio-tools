# /blog-outline

Create a detailed blog post outline.

## What this does
Develops a complete blog outline with headline options, section structure, key points, and SEO considerations - ready to write or brief a writer.

## When to use
- Starting a new blog post
- Briefing a writer or content team
- Need to organize your thoughts
- Want SEO-optimized structure

## How to use
Share:
- Topic or working title
- Target audience
- Goal of the piece (educate, convert, rank)
- Target keywords (if known)
- Approximate length

## Example
```
/blog-outline

Topic: How to improve employee onboarding
Audience: HR managers at mid-size companies
Goal: Rank for "employee onboarding best practices" + capture leads
Keywords: employee onboarding, onboarding checklist, new hire onboarding
Length: ~1500 words
```

## What you'll get
1. **3 headline options** - SEO-friendly but engaging
2. **Meta description** - For search results
3. **Section structure** - H2s and H3s mapped out
4. **Key points per section** - What to cover
5. **CTA recommendation** - How to convert readers
6. **Internal/external link suggestions** - What to reference
