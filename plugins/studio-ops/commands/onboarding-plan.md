# /onboarding-plan

Create a structured onboarding plan for new hires.

## What this does
Develops a comprehensive 30-60-90 day onboarding plan with daily activities, learning goals, key meetings, and success milestones.

## When to use
- New hire starting soon
- Current onboarding feels ad hoc
- Scaling the team and need consistency
- Expanding into new roles or departments

## How to use
Share:
- Role and level (junior, mid, senior)
- Department/team
- Key responsibilities
- Tools they'll need to learn
- Important relationships to build
- Any specific challenges for this role

## Example
```
/onboarding-plan

Role: Mid-level Account Manager
Team: Account Management (6 people)
Reports to: Account Director
Key responsibilities: Managing 3-4 client accounts, day-to-day client contact,
brief writing, project coordination
Tools: Monday.com, Slack, Google Workspace, Harvest (time tracking)
Challenge: Our AM process is mostly tribal knowledge right now
```

## What you'll get
1. **Pre-start checklist** - IT setup, access, workspace preparation
2. **Week 1 schedule** - Day-by-day orientation plan
3. **30-day learning plan** - Key knowledge areas and training
4. **60-day contribution plan** - Supervised tasks and projects
5. **90-day ownership plan** - Independent responsibilities and first review
