# /budget-tracker

Build a project or department budget tracking framework.

## What this does
Creates a structured budget tracking system with categories, monitoring cadence, variance thresholds, and reporting templates.

## When to use
- Starting a new project and need budget structure
- Current budget tracking is reactive, not proactive
- Client requires detailed financial reporting
- Need to improve project profitability

## How to use
Share:
- Project or department scope
- Budget amount and timeframe
- Cost categories (labor, production, media, etc.)
- Reporting requirements (frequency, audience)
- Current pain points with budget management

## Example
```
/budget-tracker

Project: Integrated campaign for retail client
Budget: $500K over 6 months
Categories: Strategy, creative development, production (video + photo),
media buy, digital development, project management
Reporting: Monthly to client, weekly internal
Pain point: We keep going over budget on production and discovering it too late
```

## What you'll get
1. **Budget structure** - Categories and allocation framework
2. **Tracking template** - Columns, formulas, and update cadence
3. **Variance alerts** - Thresholds that trigger action
4. **Reporting template** - Client-facing and internal formats
5. **Profitability analysis** - Margin tracking and forecasting
