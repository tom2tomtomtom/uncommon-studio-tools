# /status-report

Generate a professional status report.

## What this does
Creates a clear, client-ready status report with progress updates, risks, and next steps. Also creates internal team version with more detail.

## When to use
- Weekly or bi-weekly client updates
- Executive summaries
- When things are off track and you need to communicate clearly
- Keeping stakeholders informed

## How to use
Share:
- Project name and current phase
- What got done this period
- What's blocked or at risk
- Next steps planned
- Any asks from the client

## Example
```
/status-report

Project: Brand refresh for HealthCo
Period: This week
Done: 
- Completed stakeholder interviews (8 of 8)
- First draft of messaging framework
- Competitive audit presentation delivered
Blocked: Waiting on brand assets from their team (requested last Tuesday)
Risk: Design kickoff may slip if we don't get assets by Friday
Next: Strategy presentation Monday
Ask: Need CEO availability for final presentation
```

## What you'll get
- **Client-facing report** - Professional, clear, right level of detail
- **Internal version** - More detail for your team
- **Risk callouts** - Highlighted appropriately
- **Next steps** - Clear action items with owners
- **Escalation notes** - If needed for leadership
