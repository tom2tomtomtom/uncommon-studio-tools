# /seo-brief

Create an SEO-optimized content brief.

## What this does
Develops a comprehensive content brief with target keywords, search intent analysis, content structure, and optimization guidelines for organic search performance.

## When to use
- Planning new content that needs to rank
- Optimizing existing content that's underperforming
- Building a content strategy around specific topics
- Briefing writers on SEO requirements

## How to use
Share:
- Target topic or keyword
- Your website/brand context
- Target audience
- Competitors in this space
- Any existing content on the topic

## Example
```
/seo-brief

Topic: "project management for creative agencies"
Brand: SaaS tool built specifically for agencies
Competitors: Monday, Asana, Basecamp
Audience: Agency owners and operations managers
We have a basic blog post on this topic that gets ~50 visits/month
```

## What you'll get
1. **Primary + secondary keywords** - Target terms with search intent
2. **Content structure** - Recommended headings and sections
3. **Competitive analysis** - What's ranking and gaps to exploit
4. **On-page optimization** - Title, meta, headers, internal links
5. **Content guidelines** - Word count, tone, unique angles
