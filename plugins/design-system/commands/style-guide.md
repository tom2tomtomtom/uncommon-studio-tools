# /style-guide

Develop a comprehensive design style guide.

## What this does
Creates a detailed style guide covering typography, color palette, spacing, components, and usage guidelines that keep design work consistent.

## When to use
- Starting a new brand or product
- Current design work feels inconsistent
- Onboarding new designers to a project
- Documenting an existing but unwritten design system

## How to use
Share:
- Brand personality and values
- Existing brand assets (colors, fonts, logo)
- Target audience and industry
- Platforms the design will live on (web, mobile, print)

## Example
```
/style-guide

Brand: Health & wellness app for busy professionals
Personality: Calm, trustworthy, modern, minimal
Existing: We have a logo (teal green mark) and use Inter for our font
Platforms: iOS app, marketing website, email templates
Audience: 25-45 professionals who value efficiency
```

## What you'll get
1. **Color palette** - Primary, secondary, accent, semantic colors with hex values
2. **Typography system** - Scale, weights, line heights, and pairing rationale
3. **Spacing system** - Base unit and scale for consistent rhythm
4. **Component patterns** - Buttons, cards, inputs, navigation
5. **Usage guidelines** - Dos, don'ts, and examples
