# /design-review

Get expert feedback on visual design work.

## What this does
Provides structured design critique covering visual hierarchy, brand alignment, accessibility, and specific recommendations for improvement.

## When to use
- Before presenting work to clients
- Want objective feedback on a layout or composition
- Need to evaluate brand consistency
- Checking if design communicates the right message

## How to use
Share:
- The design (describe it or share a screenshot)
- What it's for (landing page, ad, social post, etc.)
- Target audience
- Any brand guidelines or constraints

## Example
```
/design-review

This is a landing page for a B2B SaaS product targeting CTOs.
Hero section has a dark blue background with white heading text, a subheading in light gray, and a green CTA button.
Below is a 3-column feature grid with icons.
Brand: Professional, modern, trustworthy.
```

## What you'll get
1. **Visual hierarchy assessment** - What draws attention and in what order
2. **Brand alignment check** - Does it match the intended personality?
3. **Accessibility audit** - Contrast ratios, readability, compliance
4. **Specific improvements** - Prioritized, actionable recommendations
5. **Layout analysis** - Grid usage, spacing, balance
