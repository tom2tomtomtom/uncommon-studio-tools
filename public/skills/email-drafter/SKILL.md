# Email Drafter

Write emails that get read, get responses, and get results. Clear, professional, human.

## Alt/Shift Email Voice

Professional but warm. Direct but not blunt. Helpful and action-oriented. Write like a smart colleague, not a corporate bot.

## Email Templates by Scenario

### Media Pitch Email

```
Subject: [Hook — 8 words max]

Hi [First name],

[One sentence: the news/story angle]

[One sentence: why it matters / why now]

[One sentence: what you can offer — interview, data, exclusive]

Happy to send more detail or jump on a quick call.

[Name]
[Title], Alt/Shift
[Mobile]
```

### Media Follow-Up

**First follow-up** (3-5 days after pitch):
```
Subject: Re: [Original subject]

Hi [First name],

Just checking this landed — wanted to make sure it didn't get lost in the inbox.

[One sentence restating the hook/offer]

Let me know if you'd like more info or if it's not a fit for your current focus.

[Name]
```

**Second follow-up** (1 week later):
```
Subject: Re: [Original subject]

Hi [First name],

Quick nudge on this — happy to tailor the angle if there's a different direction that works for you.

Otherwise no worries, I'll leave this with you.

[Name]
```

### Client Email — Seeking Approval

```
Subject: [Item] — For Approval by [Date]

Hi [Name],

Attached is [what you're sending] for your review.

KEY POINTS:
• [Highlight 1]
• [Highlight 2]
• [What you need them to focus on]

Please let me know by [date/time] if you're happy to proceed or if you'd like any changes.

Happy to jump on a quick call if helpful.

[Name]
```

### Introduction Email

**Introducing two people**:
```
Subject: Intro: [Name] ↔ [Name]

Hi [Name 1] and [Name 2],

I wanted to connect you two.

[Name 1] — [One sentence: who they are and why Name 2 should care]

[Name 2] — [One sentence: who they are and why Name 1 should care]

I thought you'd have plenty to discuss around [shared interest/topic].

I'll leave you to take it from here.

[Name]
```

**Warm intro request**:
```
Subject: Quick intro request

Hi [Name],

I'm hoping you might be able to connect me with [Person] at [Company].

[Why: One sentence on purpose of connection]

[Context: One sentence on why you think they could help]

Would you be comfortable making an intro? Happy to draft something if helpful.

Thanks,
[Name]
```

### Declining an Opportunity

**Declining media opportunity**:
```
Subject: Re: [Their subject]

Hi [Name],

Thanks for thinking of [Client] for this.

Unfortunately, [brief reason — timing doesn't work / not aligned with current priorities / spokesperson unavailable].

Happy to stay in touch for future opportunities — [offer alternative if appropriate].

[Name]
```

**Declining a meeting/request**:
```
Subject: Re: [Their subject]

Hi [Name],

Thanks for reaching out.

Unfortunately I'm not able to help with this right now — [brief reason if appropriate, or just leave].

[If appropriate: Suggest alternative — different person, different timing, different approach]

Appreciate you thinking of me.

[Name]
```

### Thank You Emails

**Post-meeting thank you**:
```
Subject: Thanks for today

Hi [Name],

Thanks for taking the time to meet [today/yesterday].

[One sentence: Key takeaway or agreement from meeting]

[One sentence: Confirming next steps]

Looking forward to [what comes next].

[Name]
```

**Post-coverage thank you**:
```
Subject: Thanks for the coverage

Hi [Name],

Just wanted to say thanks for the piece on [topic].

[One sentence: Why it mattered / what client appreciated]

Really appreciate the coverage. Let me know if there's anything else I can help with.

[Name]
```

### Chaser Emails

**Chasing approval**:
```
Subject: Re: [Item] — reminder

Hi [Name],

Just a gentle reminder on [what you're waiting for] — hoping to have your feedback by [deadline] so we can [why timing matters].

Let me know if you need anything from me to help.

[Name]
```

**Chasing information/assets**:
```
Subject: [What you need] — reminder

Hi [Name],

Following up on [what you're waiting for].

We need this by [date] to [consequence/reason].

Can you let me know when to expect it?

[Name]
```

## Email Principles

### Subject Lines
- Specific and scannable
- No "Quick question" or "Touching base"
- Front-load the important word
- Include client/project name where useful

### Opening Lines
- Get to the point — no "Hope this finds you well" (unless genuine relationship)
- State purpose in first sentence
- Reference context if replying

### Body
- One idea per paragraph
- Shortest version that includes all necessary info
- Bullet points for multiple items
- Bold key dates/asks if long email

### Closing
- Clear next step
- Specific deadline if needed
- Offer to help/clarify
- Appropriate sign-off

### Sign-Offs

**Professional**: Best regards, Thanks, Kind regards
**Warmer**: Thanks, Cheers, Speak soon
**Quick exchanges**: [Just name], [Initial]

## Formatting Standards

- Short paragraphs (2-3 sentences max)
- Line break between paragraphs
- Bullet points for lists
- Bold for emphasis sparingly
- No coloured text
- Standard fonts
- Include mobile in signature

## Quality Checklist

- [ ] Subject line specific and scannable
- [ ] Opens with purpose
- [ ] Key ask clear
- [ ] Deadline specified (if needed)
- [ ] Tone matches relationship
- [ ] Shortest version possible
- [ ] Proofread for errors
- [ ] Right people CC'd
- [ ] Attachments attached
